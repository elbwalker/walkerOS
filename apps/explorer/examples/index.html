Quick Start

To use the new Explorer components in your project:

ES Modules (Recommended)

// LiveCode component
import { createLiveCode } from '@walkeros/explorer';

const liveCode = createLiveCode(element, {
  input: 'return 1 + 1;',
  context: { /* your functions */ }
});

// Playground component
import { createPlayground } from '@walkeros/explorer';

const playground = createPlayground(element, {
  initialHTML: '<div data-elb="product">...</div>',
  initialMapping: { /* mapping rules */ }
});

Architecture Overview

The new Explorer follows atomic design principles and includes a graph-based event flow system:

Component Hierarchy

  • Atoms: Basic building blocks (box, editor, button, label)
  • Molecules: Composed components (codeBox, resultBox)
  • Organisms: Complete features (liveCode, mapping, playground)
  • Layouts: Responsive layout systems

Graph System

  • Nodes: HTMLNode, PreviewNode, CollectorNode, MappingNode, DestinationNode, ConsoleNode
  • Edges: Define data flow between nodes
  • Engine: DAG validation and topological execution
