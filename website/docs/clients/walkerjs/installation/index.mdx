import Link from '@docusaurus/Link';

There are multiple ways to install and use the web client walker.js. Each one
has their own advantages.

- <Link to="cdn">**CDN**</Link>: Load walker.js from a CDN via script tag.
- <Link to="gtm">**Google Tag Manager**</Link>: Use the GTM to load walker.js.
- <Link to="NPM">**NPM**</Link>: Install walker.js as an NPM package.
- <Link to="script">**Code**</Link>: Directly include walker.js in the browser.
