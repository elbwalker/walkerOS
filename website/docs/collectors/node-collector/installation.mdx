Install walkerOS node source via NPM:

```bash
npm install @walkerOS/node-collector
```

Use the `createNodeCollector` function to create a new collector instance. It returns
the `elb` function to
use&nbsp;<Link to="/docs/collectors/node-collector/commands">commands</Link>, pushing events,
and the `instance` itself.

```ts
import { createNodeCollector } from '@walkerOS/node-collector';
const { elb, instance } = createNodeCollector({});
```
