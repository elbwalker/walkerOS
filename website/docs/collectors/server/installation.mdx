Install walkerOS server source via NPM:

```bash
npm install @walkerOS/server-collector
```

Use the `createServerCollector` function to create a new collector instance. It returns
the `elb` function to
use&nbsp;<Link to="/docs/collectors/server/commands">commands</Link>, pushing events,
and the `instance` itself.

```ts
import { createServerCollector } from '@walkerOS/server-collector';
const { elb, instance } = createServerCollector({});
```
