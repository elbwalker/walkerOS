# GCP Server Package

The `@walkerOS/server-destination-gcp` package provides BigQuery integration for server applications:

## Destinations

- **[BigQuery](./bigquery)** - Send events to Google BigQuery for data warehousing

## Installation

```bash
npm install @walkerOS/server-destination-gcp
```

## Basic Usage

```javascript
import { elb } from '@walkerOS/server-collector';
import { destinationBigQuery } from '@walkerOS/server-destination-gcp';

// Add BigQuery destination
elb('walker destination', destinationBigQuery, {
  projectId: 'your-project-id',
  datasetId: 'your-dataset-id',
  tableId: 'your-table-id'
});
```
