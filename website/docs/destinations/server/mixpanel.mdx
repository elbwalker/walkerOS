# Mixpanel

<PackageLink env="server" github="packages/server/destinations/mixpanel" npm="@walkeros/server-destination-mixpanel" />

Server-side event delivery to Mixpanel for product analytics, people profiles, and group analytics via the official [`mixpanel`](https://www.npmjs.com/package/mixpanel) Node.js SDK.

:::info Where This Fits
Mixpanel is a **server destination** in the walkerOS flow:

<FlowMap
  sources={{ default: { highlight: false } }}
  collector={{ highlight: false }}
  destinations={{ default: { label: 'Destination', text: 'Mixpanel', highlight: true } }}
/>

Sends events server-side to Mixpanel via `/track` (real-time, up to 5 days old) or `/import` (historical, any age). Supports people profile operations, group analytics, and identity resolution.
:::

## Installation

<CodeSnippet
  code={`npm install @walkeros/server-destination-mixpanel`}
  language="bash"
/>
