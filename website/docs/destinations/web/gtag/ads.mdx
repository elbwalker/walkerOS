# Google Ads

<PackageLink

[Google Ads conversion tracking](https://support.google.com/google-ads/answer/1722022?hl=en)
enables you to track conversions and measure the effectiveness of your Google
Ads campaigns via the unified gtag destination.

<CodeSnippet

/>
