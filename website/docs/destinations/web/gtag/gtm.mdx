# Google Tag Manager (GTM)

<PackageLink
  env="web"
  github="packages/web/destinations/gtag"
  npm="@walkeros/web-destination-gtag"
/>

[Google Tag Manager (GTM)](https://tagmanager.google.com/) is a tool to manage
and add third-party integrations to your website, configured within the
unified gtag destination.

## Installation

<CodeSnippet
  code={`npm install @walkeros/web-destination-gtag`}
  language="bash"
/>

<CodeSnippet
  code={`import { startFlow } from '@walkeros/collector';
import { destinationGtag } from '@walkeros/web-destination-gtag';

await startFlow({
  destinations: {
    gtag: {
      code: destinationGtag,
      config: {
        settings: {
          gtm: {
            containerId: 'GTM-XXXXXXX',
            dataLayer: 'dataLayer',
          },
        },
      },
    },
  },
});`}
  language="typescript"
/>
