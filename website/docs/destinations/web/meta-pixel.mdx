# Meta Pixel

<PackageLink
  env="web"
  github="packages/web/destinations/meta"
  npm="@walkeros/web-destination-meta"
/>

Send events to Meta Pixel (formerly Facebook Pixel) to track visitor activity
and conversions for Facebook and Instagram advertising campaigns.

## Installation

```bash
npm install @walkeros/web-destination-meta
```

## Setup

```typescript
import { createCollector } from '@walkeros/collector';
import { destinationMeta } from '@walkeros/web-destination-meta';

const { collector, elb } = await createCollector({
  destinations: {
    meta: {
      code: destinationMeta,
      config: {
        settings: {
          pixelId: '1234567890', // Your Meta Pixel ID
        },
        loadScript: true, // Load Meta Pixel script automatically
      },
    },
  },
});
```
