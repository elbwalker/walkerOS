# Plausible Analytics

<PackageLink
  env="web"
  github="packages/web/destinations/plausible"
  npm="@walkeros/web-destination-plausible"
/>

[Plausible Analytics](https://plausible.io/) is a simple, privacy-friendly
Google Analytics alternative. The destination loads the Plausible script,
initializes the `plausible` function, and sends page views and custom events.

<WhereThisFits
  pre={<>Plausible is a <strong>web destination</strong> in the walkerOS flow:</>}
  flowMap={{
    destinations: { default: { label: 'Destination', text: 'Plausible', highlight: true } },
  }}
/>
