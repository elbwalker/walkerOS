# One event, four tools

This web flow sends one event stream to four analytics tools at once, with no per-tool mapping. The collector fans a single event out to every destination, and each tool receives it in its own native call shape.

## What each tool receives

With no mapping, each destination forwards the event under its own default. For an `order complete` event:

- GA4: `gtag('event', 'order_complete', { send_to: 'G-XXXXXX-1' })`
- Amplitude: `amplitude.track('order complete', {})`
- Mixpanel: `mixpanel.track('order complete', {})`
- PostHog: `posthog.capture('order complete', {})`

The three product-analytics tools keep the walkerOS event name and send empty properties. GA4 lowercases the name and always adds `send_to`.

To reshape events per tool, see [GA4 ecommerce](/docs/getting-started/ga4-ecommerce).
