# Quickstart Examples

Get started with walkerOS in minutes. These examples provide working code that
you can copy, paste, and run immediately.

## Detailed Examples

### Core Setup

#### Basic Collector

**What it demonstrates:** Fundamental event collection using the walkerOS
collector without any sources or destinations.

**Try it yourself:**

<Tabs>
  <TabItem value="npm" label="NPM" default>

```bash
npm install @walkeros/collector
```

Copy the code from
[collector/basic.ts](https://github.com/elbwalker/walkerOS/blob/main/apps/quickstart/src/collector/basic.ts),
run it, and see events logged to console.

  </TabItem>
  <TabItem value="browser" label="Browser/CDN">

```html
<script src="https://cdn.jsdelivr.net/npm/@walkeros/collector@latest/dist/index.browser.js"></script>
<script>
  const { createCollector } = walkerOSCollector;

  async function setup() {
    const { collector, elb } = await createCollector({
      destinations: {
        console: { push: (event) => console.log('Event:', event) },
      },
    });

    await elb('page view', { title: document.title });
  }

  setup();
</script>
```

  </TabItem>
</Tabs>

### Web Sources

#### Browser Tracking

**What it demonstrates:** Automatic DOM event tracking for clicks, views, and
form submissions using the browser source.

**Try it yourself:**

<Tabs>
  <TabItem value="npm" label="NPM" default>

```bash
npm install @walkeros/collector @walkeros/web-source-browser
```

Copy
[web-browser/basic.ts](https://github.com/elbwalker/walkerOS/blob/main/apps/quickstart/src/web-browser/basic.ts),
add `data-elb` attributes to your HTML elements, and see automatic tracking.

  </TabItem>
  <TabItem value="browser" label="Browser/CDN">

```html
<script src="https://cdn.jsdelivr.net/npm/@walkeros/collector@latest/dist/index.browser.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@walkeros/web-source-browser@latest/dist/index.browser.js"></script>
<script>
  async function setup() {
    const { createCollector } = walkerOSCollector;
    const { sourceBrowser } = walkerOSWebSourceBrowser;

    const { collector, elb } = await createCollector({
      sources: {
        browser: {
          code: sourceBrowser,
          config: {
            settings: {},
          },
        },
      },
      destinations: {
        console: { push: (event) => console.log('Browser Event:', event) },
      },
    });
  }

  setup();
</script>

<!-- Add tracking attributes -->
<div data-elb="product" data-elbaction="view" data-elbid="prod-123">
  Product Details
</div>
```

  </TabItem>
</Tabs>

### Web Destinations

#### Google Analytics 4

**What it demonstrates:** Complete GA4 integration with page views, e-commerce
events, and custom parameters.

**Try it yourself:**

<Tabs>
  <TabItem value="npm" label="NPM" default>

```bash
npm install @walkeros/collector @walkeros/web-destination-gtag
```

Copy
[web-destinations/ga4-complete.ts](https://github.com/elbwalker/walkerOS/blob/main/apps/quickstart/src/web-destinations/ga4-complete.ts),
add your GA4 measurement ID, and see events in GA4 real-time reports.

  </TabItem>
  <TabItem value="browser" label="Browser/CDN">

```html
<script
  async
  src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"
></script>
<script src="https://cdn.jsdelivr.net/npm/@walkeros/collector@latest/dist/index.browser.js"></script>
<script>
  // Initialize GA4
  window.dataLayer = window.dataLayer || [];
  function gtag() {
    dataLayer.push(arguments);
  }
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');

  // Setup walkerOS with GA4 destination
  async function setup() {
    const { createCollector } = walkerOSCollector;

    const { collector, elb } = await createCollector({
      destinations: {
        ga4: {
          push: (event) => {
            gtag('event', event.event.replace(' ', '_'), {
              event_category: event.entity,
              event_label: event.action,
              value: event.data.value,
              custom_parameters: event.data,
            });
          },
        },
      },
    });

    // Track page view
    await elb('page view', { title: document.title });
  }

  setup();
</script>
```

</Tabs>

#### Custom Destination

**What it demonstrates:** Building your own destination to send events to any
service via webhooks or APIs.

**Try it yourself:**

<Tabs>
  <TabItem value="npm" label="NPM" default>

```bash
npm install @walkeros/collector
```

Copy
[web-destinations/custom-destination.ts](https://github.com/elbwalker/walkerOS/blob/main/apps/quickstart/src/web-destinations/custom-destination.ts),
modify the webhook URL, and create your own integration.

  </TabItem>
  <TabItem value="browser" label="Browser/CDN">

```html
<script src="https://cdn.jsdelivr.net/npm/@walkeros/collector@latest/dist/index.browser.js"></script>
<script>
  async function setup() {
    const { createCollector } = walkerOSCollector;

    const { collector, elb } = await createCollector({
      destinations: {
        webhook: {
          push: async (event) => {
            await fetch('https://webhook.site/your-unique-id', {
              method: 'POST',
              headers: { 'Content-Type': 'application/json' },
              body: JSON.stringify({
                event: event.event,
                data: event.data,
                timestamp: new Date().toISOString(),
              }),
            });
          },
        },
      },
    });

    await elb('custom event', { message: 'Hello from browser!' });
  }

  setup();
</script>
```

</Tabs>

#### Consent Management

**What it demonstrates:** GDPR/CCPA compliant tracking with consent state
management and conditional event sending.

**Try it yourself:**

<Tabs>
  <TabItem value="npm" label="NPM" default>

```bash
npm install @walkeros/collector @walkeros/web-destination-gtag
```

Copy
[consent/management.ts](https://github.com/elbwalker/walkerOS/blob/main/apps/quickstart/src/consent/management.ts)
and implement compliant user tracking with consent controls.

  </TabItem>
  <TabItem value="browser" label="Browser/CDN">

```html
<script src="https://cdn.jsdelivr.net/npm/@walkeros/collector@latest/dist/index.browser.js"></script>
<script>
  async function setup() {
    const { createCollector } = walkerOSCollector;

    const { collector, elb } = await createCollector({
      destinations: {
        console: { push: (event) => console.log('Consent Event:', event) },
      },
    });

    // Initially disable tracking
    collector.allowed = false;

    // Simulate consent acceptance
    setTimeout(async () => {
      await elb('walker consent', {
        functional: true,
        analytics: true,
        marketing: false,
      });
      collector.allowed = true;

      // Now events will be sent
      await elb('page view', { title: 'Consented Page' });
    }, 2000);
  }

  setup();
</script>
```
