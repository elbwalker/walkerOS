# Quickstart

Get started with walkerOS in minutes. These examples provide working code that
you can copy, paste, and run immediately. All examples are maintained in our
[quickstart app](https://github.com/elbwalker/walkerOS/tree/main/apps/quickstart)
and tested with every release.

## 1. Install packages

Install the collector package from npm:

```bash
npm install @walkeros/collector @walkeros/web-source-browser
```

## 2. Create basic walkerOS setup

```javascript
import type { Collector, WalkerOS } from '@walkeros/core';
import { createCollector } from '@walkeros/collector';
import { createSource } from '@walkeros/core';
import { sourceBrowser } from '@walkeros/web-source-browser';

// Initialize walkerOS
export async function initializeWalker(): Promise<void> {
  const { collector, elb } = await createCollector({
    destinations: {
      console: {
        push: (event) => console.log('Event:', event),
      },
    },
    sources: {
      browser: createSource(sourceBrowser, {
        settings: {
          prefix: 'data-elb',
          pageview: true,
          session: true,
        },
      }),
    },
  });

  // Make available globally (optional)
  window.walker = collector;
  window.elb = elb;
}
```

## 3. Add destinations

Install destination packages and add them to your setup:

```javascript
import { createCollector } from '@walkeros/collector';
import { createSource } from '@walkeros/core';
import { sourceBrowser } from '@walkeros/web-source-browser';
import { destinationGtag } from '@walkeros/web-destination-gtag';
import { destinationAPI } from '@walkeros/web-destination-api';

export async function initializeWalker(): Promise<void> {
  const { collector, elb } = await createCollector({
    destinations: {
      // Send events to your API
      api: {
        ...destinationAPI,
        config: {
          settings: {
            url: 'https://your-api.com/events',
          },
        },
      },

      // Send to Google Analytics 4
      ga4: {
        ...destinationGtag,
        config: {
          settings: {
            ga4: { measurementId: 'G-XXXXXXXXXX' },
          },
        },
      },
    },
    sources: {
      browser: createSource(sourceBrowser, {
        settings: {
          prefix: 'data-elb',
          pageview: true,
          session: true,
        },
      }),
    },
  });

  window.walker = collector;
  window.elb = elb;
}
```

## 4. Add consent management

Add consent requirements to your destinations:

```javascript
export async function initializeWalker(): Promise<void> {
  const { collector, elb } = await createCollector({
    destinations: {
      // API destination - requires functional consent
      api: {
        ...destinationAPI,
        config: {
          settings: {
            url: 'https://your-api.com/events',
          },
          consent: {
            functional: true,
          },
        },
      },

      // GA4 - requires e.g. analytics consent
      ga4: {
        ...destinationGtag,
        config: {
          settings: {
            ga4: { measurementId: 'G-XXXXXXXXXX' },
          },
          consent: {
            analytics: true,
          },
        },
      },
    },
    sources: {
      browser: createSource(sourceBrowser, {
        settings: {
          prefix: 'data-elb',
          pageview: true,
          session: true,
        },
      }),
    },
  });

  window.walker = collector;
  window.elb = elb;

  // Set consent when user accepts/denies
  // This would typically come from your consent banner
  function handleConsentAccept() {
    elb('walker consent', {
      functional: true,
      analytics: true,
      marketing: true,
    });
  }

  function handleConsentDeny() {
    elb('walker consent', {
      functional: true,
      analytics: false,
      marketing: false,
    });
  }
}
```

```typescript
const {
  collector,
  elb,

  collector: Collector.Instance;

  // Configuration with full type safety
});
```

## Advanced Options

### Version Pinning

Pin to specific versions for production stability:

```bash
# NPM version pinning
npm install @walkeros/collector@0.0.7 @walkeros/web-source-browser@0.0.7

```

For CDN usage with dynamic imports:

```javascript
// Pin to specific versions const { createCollector } = await import(
'https://cdn.jsdelivr.net/npm/@walkeros/collector@0.0.7/dist/index.mjs' ); const
{ sourceBrowser } = await import(
'https://cdn.jsdelivr.net/npm/@walkeros/web-source-browser@0.0.7/dist/index.mjs'
);
```

### Self-Hosting

Download and host the packages yourself:

```bash
# Download specific package
# Download specific packages
npm pack @walkeros/collector
npm pack @walkeros/web-source-browser

# Extract files
tar -xzf walkeros-collector-*.tgz
tar -xzf walkeros-web-source-browser-*.tgz

# Copy to your public directory
cp collector-package/dist/* ./public/js/collector/
cp browser-package/dist/* ./public/js/browser/
```

```html
// Use self-hosted files const { createCollector } = await
import('/js/collector/index.mjs'); const { sourceBrowser } = await
import('/js/browser/index.mjs');
```
