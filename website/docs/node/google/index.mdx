# Google Node Package

The `@walkerOS/node-google` package provides Google integrations for Node.js applications:

## Destinations

- **[BigQuery](./bigquery)** - Send events to Google BigQuery for data warehousing

## Installation

```bash
npm install @walkerOS/node-google
```

## Basic Usage

```javascript
import { elb } from '@walkerOS/node';
import { destinationBigQuery } from '@walkerOS/node-google';

// Add BigQuery destination
elb('walker destination', destinationBigQuery, {
  projectId: 'your-project-id',
  datasetId: 'your-dataset-id',
  tableId: 'your-table-id'
});
```
