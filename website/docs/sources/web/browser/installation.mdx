# Installation

<Tabs>
  <TabItem value="npm" label="npm" default>
Install the source via npm

```bash
npm install @walkerOS/web-source-browser
```

```js
import { elb } from '@walkerOS/web-source-browser';

elb('walker source', config);
```

</TabItem>
<TabItem value="script" label="script">
Load the destination via dynamic import

```html
<script>
  // Upload the dist/index.mjs on your own server
  const source = (
    await import(
      'https://cdn.jsdelivr.net/npm/@walkerOS/web-source-browser/dist/index.mjs'
    )
  ).sourceBrowser;

  elb('walker source', source, config);
</script>
```
