Utils are included in the collectors. For most use cases, install the collector
instead:

- **<Link to="/docs/collectors/web/installation/">Web Collector</Link>** -
  includes web and core utils
- **<Link to="/docs/collectors/server/installation/">Server Collector</Link>** -
  includes server and core utils

## Standalone Installation

For advanced use cases requiring only specific utils:

### NPM

```bash
$ npm install @walkerOS/utils
```

### CDN

There are multiple build files available. Bundled with all functions, grouped
only for the core or web, and separated into individual files. The `es5` files
can be used in older browsers as well as with Google Tag Manager.

<Tabs>
  <TabItem value="script" label="Scripts" default>
    ```html
    <script src="https://cdn.jsdelivr.net/npm/@walkerOS/utils@latest/dist/web.js"></script>
    ```
  </TabItem>
  <TabItem value="module" label="Module">
    Load the `.mjs` module-file directly from the CDN or your own server.
    ```html
    <script type="module">
      const Utils = ( await import(
      'https://cdn.jsdelivr.net/npm/@walkerOS/utils@latest/dist/web.mjs' ) );
      console.log('id', Utils.getId())
    </script>
    ```
  </TabItem>
</Tabs>

:::warning

Be aware of the `latest` version in the URL, better use a specific version.

:::
