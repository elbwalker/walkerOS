[Google Ads](https://ads.google.com/) is Google's comprehensive advertising
platform. The walker.js Google Ads destination enables seamless integration of
your web application with Google Ads.

## Installation

<Tabs>
  <TabItem value="ts" label="TypeScript" default>
Install the destination via npm

```ts
import { elb } from '@walkerOS/web';
import { destinationAds } from '@walkerOS/web-google';

elb('walker destination', destinationAds, config);
```

  </TabItem>
  <TabItem value="script" label="Script">

Load the destination via dynamic import

```html
<script>
  // Upload the dist/index.mjs on your own server
  const destination = (
    await import(
      'https://cdn.jsdelivr.net/npm/@walkerOS/web-google/dist/index.mjs'
    )
  ).destinationAds;

  elb('walker destination', destination, config);
</script>
```

  </TabItem>

## Configuration

Configure the Google Ads destination with a Conversion ID and Conversion label.

<DestinationInit
  custom={`{
    conversionId: 'AW-123456789', // Required
    currency: 'EUR',
    defaultValue: 1,
  }`}
/>

### Custom

| Property       | Type   | Description                                      |
| -------------- | ------ | ------------------------------------------------ |
| conversionId\* | string | Google Ads conversion ID, starting with 'AW-'' . |
| currency       | string | Default currency for conversions.                |
| defaultValue   | number | Default value used for conversions.              |

Use `name` to specify the conversion label and `data` for other parameters.
